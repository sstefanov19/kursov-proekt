## 5. Проектиране

### 5.1 Потребители

- Ученик / играч
- Потенциално teacher / classroom organizer

### 5.2 Основни функционалности

- регистрация и вход
- игра с математически въпроси
- XP и level progression
- perk система
- глобална класация
- classroom стаи и classroom leaderboard

### 5.3 Архитектура

- Frontend: React Native + Expo + TypeScript
- Backend: Spring Boot + Java 21
- Database: PostgreSQL
- Auth: JWT
