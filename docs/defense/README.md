# Материали за защита

Тази папка съдържа готово съдържание за:

- 5-минутна PowerPoint презентация
- 3-минутно демо
- списък със screenshot-и за слайдовете
- кратка подготовка за въпроси от комисия

## Файлове

- `presentation-slides.md` — съдържание по слайдове, включително говорни бележки
- `demo-script.md` — стъпки за живо демо и backup видео
- `screenshots-checklist.md` — какви екрани да се заснемат
- `committee-qa.md` — кратки отговори на очаквани въпроси

## Практически указания

1. Направете PowerPoint с 7-8 слайда по `presentation-slides.md`.
2. Добавете реални screenshot-и от приложението на местата, описани в checklist-а.
3. Репетирайте по `demo-script.md`, докато стигнете 5 мин. презентация + 3 мин. демо.
4. Запишете демо видео като резервен вариант при проблем с интернет, база данни или устройство.
